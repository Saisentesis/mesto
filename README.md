# Проект: Место

### Обзор

* Описание
* Ссылки

**Описание**

Сервис mesto для хранения фотографий.

* Сайт адаптирован под разрешения 1280 и 320px.

* Использованы обращения к DOM-дереву на Js.

* Файловая структура организована БЭМ.
